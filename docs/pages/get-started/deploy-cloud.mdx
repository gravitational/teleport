Teleport Enterprise Cloud is the **fastest and easiest way to get started**.  

You don't need to set up infrastructure, manage certificates, or install anything. We handle all of that for you.

All you need to do is sign up for a 14-day free trial. We'll automatically provision and manage a fully featured Teleport cluster in a dedicated tenant with a unique `example.teleport.sh` domain.

**To deploy your cluster:**

1. Go to [goteleport.com/signup](https://goteleport.com/signup)
2. Follow the sign-up steps to launch your cluster

That's it. No setup scripts, no infrastructure provisioning, no config files.

You now have a production-grade Teleport control plane with secure defaults, automatic updates, and built-in scalability.
