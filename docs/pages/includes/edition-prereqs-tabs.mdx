<TabItem scope={["oss"]} label="Teleport Community Edition">

- A running Teleport cluster. For details on how to set this up, see the
  [Getting Started](../index.mdx) guide.

- The `tctl` admin tool and `tsh` client tool version >= (=teleport.version=).

  See [Installation](../installation.mdx) for details.

To check version information, run the `tctl version` and `tsh version` commands.
For example:

```code
$ tctl version
# Teleport v(=teleport.version=) git:(=teleport.git=) go(=teleport.golang=)
  
$ tsh version
# Teleport v(=teleport.version=) go(=teleport.golang=)
Proxy version: (=teleport.version=)
Proxy: (=teleport.url=)
```

</TabItem>
