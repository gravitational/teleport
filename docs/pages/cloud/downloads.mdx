Teleport Cloud is currently running Teleport (=cloud.version=). We recommend
running Teleport agents at Teleport (=cloud.version=) or one major version
below.

## Teleport archives

The following archives contain all of the binaries you will need manually
install and run Teleport.

Binaries include:

|Binary|Purpose|
|---|---|
|`teleport`|The Teleport binary that you can run within your private networks to enable access to resources in your infrastructure|
|`tctl`| A client tool for managing Teleport resources|
|`tsh`| A client tool for accessing resources within your cluster|
|`tbot`| A client tool you will use to associate a bot user with a machine identity|

Here are the available archives:

Description                        | | | | |
-----------------------------------|-|-|-|-|
Linux (Debian/Ubuntu compatible)   | [amd64](https://get.gravitational.com/teleport-ent-v(=cloud.version=)-linux-amd64-bin.tar.gz) | [arm64](https://get.gravitational.com/teleport-ent-v(=cloud.version=)-linux-arm64-bin.tar.gz) | [armv7](https://get.gravitational.com/teleport-ent-v(=cloud.version=)-linux-arm-bin.tar.gz) | [i386](https://get.gravitational.com/teleport-ent-v(=cloud.version=)-linux-386-bin.tar.gz) |
Linux (RHEL/CentOS 7.x compatible) | [amd64](https://get.gravitational.com/teleport-ent-v(=cloud.version=)-linux-amd64-centos7-bin.tar.gz) |
macOS                              | [amd64](https://get.gravitational.com/teleport-ent-v(=cloud.version=)-darwin-amd64-bin.tar.gz) |
