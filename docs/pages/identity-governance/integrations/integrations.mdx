You can configure Teleport to sync role-based access control (RBAC) configurations with
third-party software, such as AWS IAM Identity Center, Microsoft Entra ID, and
Okta. Teleport creates resources such as roles and Access Lists based on
resources it retrieves from the integration. This way, you can manage all access
controls in your infrastructure from a single platform.

<Admonition type="tip" title="Review imported access">
Once an integration imports resources into Teleport, you can use Teleport
Identity Security to review which identities can access them and whether that
access is used. See [Review Identity Access from the
CLI](../../identity-security/cli/access-review.mdx) (`tctl access-review`).
</Admonition>
