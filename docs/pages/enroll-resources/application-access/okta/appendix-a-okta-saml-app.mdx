This page will guide you how to create a new Okta SAML 2.0 app for *Level 1. SSO* integration. It
is also used as source of users for *Level 2. SCIM* and *Level 3. User sync* integration.

<Admonition type="note">
If you have exiting working Teleport Auth Connector, you'd like to reuse, then you are all set and
can skip this section.
</Admonition>
