## Inject JWT

(!docs/pages/includes/application-access/jwt-inject.mdx!)

For example:

```yaml
- name: "elasticsearch"
  uri: https://localhost:4321
  public_addr: elastic.example.com
  rewrite:
    headers:
    - "Authorization: Bearer {{internal.jwt}}"
```
