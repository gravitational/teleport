## Inject JWT

(!docs/pages/includes/application-access/jwt-inject.mdx!)

For example:

```yaml
- name: "my-mcp-server"
  uri: mcp+http://localhost:4321
  rewrite:
    headers:
    - "Authorization: Bearer {{internal.jwt}}"
```
