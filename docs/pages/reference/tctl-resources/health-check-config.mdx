## Health Check Config Spec

The health check spec.


Example:

```yaml
match: # [...]
timeout: # See description
interval: # See description
healthy_threshold: 1
unhealthy_threshold: 1
```

|Field Name|Description|Type|
|---|---|---|
|unhealthy_threshold|The number of consecutive failing health checks after which a target's health status becomes "unhealthy".|number|
|timeout|The health check connection establishment timeout. An attempt that times out is a failed attempt.||
|match|Used to select resources that these settings apply to.|[Matcher](#matcher)|
|interval|The time between each health check.||
|healthy_threshold|The number of consecutive passing health checks after which a target's health status becomes "healthy".|number|
