## Github Connector Spec

A Github connector specification.


Example:

```yaml
client_id: "string"
client_secret: "string"
redirect_url: "string"
teams_to_logins: 
  - # [...]
  - # [...]
  - # [...]
display: "string"
teams_to_roles: 
  - # [...]
  - # [...]
  - # [...]
endpoint_url: "string"
api_endpoint_url: "string"
client_redirect_settings: # [...]
```

|Field Name|Description|Type|
|---|---|---|
|teams_to_roles|Maps Github team memberships onto allowed roles.|[][Team Roles Mapping](#team-roles-mapping)|
|teams_to_logins|Maps Github team memberships onto allowed logins/roles.  DELETE IN 11.0.0 Deprecated: use GithubTeamsToRoles instead.|[][Team Mapping](#team-mapping)|
|redirect_url|The authorization callback URL.|string|
|endpoint_url|The URL of the GitHub instance this connector is for.|string|
|display|The connector display name.|string|
|client_secret|The Github OAuth app client secret.|string|
|client_redirect_settings|Defines which client redirect URLs are allowed for non-browser SSO logins other than the standard localhost ones.|[SSOClient Redirect Settings](#ssoclient-redirect-settings)|
|client_id|The Github OAuth app client ID.|string|
|api_endpoint_url|The URL of the API endpoint of the Github instance this connector is for.|string|

```yaml
  - "string"
```

|Field Name|Description|Type|

## Team Mapping

Represents a single team membership mapping.  DELETE IN 11.0.0


Example:

```yaml
organization: "string"
team: "string"
logins: 
  - "string"
  - "string"
  - "string"
kubernetes_groups: 
  - "string"
  - "string"
  - "string"
kubernetes_users: 
  - "string"
  - "string"
  - "string"
```

|Field Name|Description|Type|
|---|---|---|
|team|A team within the organization a user belongs to.|string|
|organization|A Github organization a user belongs to.|string|
|logins|A list of allowed logins for this org/team.|[]string|
|kubernetes_users|A list of allowed kubernetes users to impersonate for this org/team.|[]string|
|kubernetes_groups|A list of allowed kubernetes groups for this org/team.|[]string|
