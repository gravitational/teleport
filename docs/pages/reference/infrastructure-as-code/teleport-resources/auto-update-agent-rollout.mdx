The auto-update agent rollout resource allows cluster administrators to view the
current agent rollout for Teleport Agent Managed Updates (v2).

This resource should not be edited by humans.

(!docs/pages/includes/reference/resources/autoupdate_agent_rollout.mdx!)

See [Teleport Agent Managed Updates](../../../upgrading/agent-managed-updates/agent-managed-updates.mdx) for more details.
