The `teleport-plugin-msteams` Helm chart is used to configure the MsTeams Teleport plugin, which allows users to receive Access Requests via channels or direct messages in MsTeams.

You can [browse the source on GitHub](https://github.com/gravitational/teleport/tree/v(=teleport.version=)/examples/chart/access/msteams).

This reference details available values for the `teleport-plugin-msteams` chart.
