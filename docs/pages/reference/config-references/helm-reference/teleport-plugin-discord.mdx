The `teleport-plugin-discord` Helm chart is used to configure the Discord Teleport plugin, which allows users to receive Access Requests via channels or direct messages in Discord.

You can [browse the source on GitHub](https://github.com/gravitational/teleport/tree/v(=teleport.version=)/examples/chart/access/discord).

This reference details available values for the `teleport-plugin-discord` chart.
