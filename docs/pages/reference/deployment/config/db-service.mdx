## Fields

Example:

```yaml

  enabled: "string"
  listen_addr: "string"
  databases: 
    - # [...]
    - # [...]
    - # [...]

    - # [...]
    - # [...]
    - # [...]

    - # [...]
    - # [...]
    - # [...]
```

## AWS Organization Units Matcher

Contains rules for matching accounts under an Organization. Accounts that belong to an excluded Organizational Unit, and its children, will be excluded even if they were included.

Example:

```yaml
include: 
  - "string"
  - "string"
  - "string"
exclude: 
  - "string"
  - "string"
  - "string"
```
|Field Name|Description|Type|
|---|---|---|
|exclude|A list of AWS Organizational Unit IDs and children OUs to exclude. Accounts that belong to these OUs, and their children, will be excluded, even if they were included. Only exact matches are supported. Optional. If empty, no OUs are excluded.|[]string|
|include|A list of AWS Organizational Unit IDs and children OUs to include. Accounts that belong to these OUs, and their children, will be included. Only exact matches or wildcard (*) are supported. Required.|[]string|
