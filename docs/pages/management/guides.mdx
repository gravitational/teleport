You can integrate Teleport with third-party tools in order to complete various
tasks in your cluster. These guides describe Teleport integrations that are not
documented elsewhere:

{/*TOPICS*/}

- [AWS OIDC Integration](guides/awsoidc-integration.mdx): How to connect your AWS account with Teleport and provide access to AWS resources.
- [EC2 Tags as Teleport Node Labels](guides/ec2-tags.mdx): How to set up Teleport Node labels based on EC2 tags
- [GCP Tags and Labels as Teleport Agent Labels](guides/gcp-tags.mdx): How to set up Teleport agent labels based on GCP tags and labels
- [Using Teleport's Certificate Authority with GitHub](guides/ssh-key-extensions.mdx): How to use Teleport's short-lived SSH certificates with the GitHub Certificate Authority.
