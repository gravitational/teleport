If you run a self-hosted Teleport cluster, using Teleport Identity Security
requires running two services on your own infrastructure:

- **Teleport Access Graph Service:** Powers access path exploration features,
  including the Dashboard, Browse, Crown Jewels, Graph Explorer, and SQL Editor
  views.
- **Teleport Identity Activity Center:** Powers audit log centralization and
  querying, including the Browse and Alerts views.
