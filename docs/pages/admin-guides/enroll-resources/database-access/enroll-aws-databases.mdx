title: Enroll AWS Databases
description: "Provides instructions on protecting databases in your AWS-managed infrastructure with Teleport."

The guides in this section show you how to protect AWS-managed databases with
Teleport.

You can configure Teleport to discover databases in your AWS account and enroll
them with your cluster automatically. Read more about [Teleport auto-discovery
for AWS databases](../auto-discovery/databases.mdx).

It is also possible to protect databases across your AWS accounts. Read the
instructions in [AWS Cross-Account Database
Access](./enroll-aws-databases/aws-cross-account.mdx).

Read the following guides for how to protect a specific AWS-managed database
with Teleport:

- [Database Access with AWS ElastiCache and AWS MemoryDB for Redis](enroll-aws-databases/redis-aws.mdx): How to configure Teleport database access with AWS ElastiCache and AWS MemoryDB for Redis.
- [Database Access with AWS OpenSearch](enroll-aws-databases/aws-opensearch.mdx): How to access AWS OpenSearch with Teleport database access
- [Database Access with AWS RDS Proxy for MariaDB/MySQL](enroll-aws-databases/rds-proxy-mysql.mdx): How to configure Teleport database access with AWS RDS Proxy for MariaDB/MySQL.
- [Database Access with AWS RDS Proxy for Microsoft SQL Server](enroll-aws-databases/rds-proxy-sqlserver.mdx): How to configure Teleport database access with AWS RDS Proxy for Microsoft SQL Server
- [Database Access with AWS RDS Proxy for PostgreSQL](enroll-aws-databases/rds-proxy-postgres.mdx): How to configure Teleport database access with AWS RDS Proxy for PostgreSQL
- [Database Access with AWS RDS and Aurora](enroll-aws-databases/rds.mdx): How to configure Teleport database access with AWS RDS and Aurora for PostgreSQL, MySQL and MariaDB.
- [Database Access with Amazon DynamoDB](enroll-aws-databases/aws-dynamodb.mdx): How to access Amazon DynamoDB with Teleport database access
