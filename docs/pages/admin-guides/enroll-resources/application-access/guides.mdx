These guides explain how to use the Teleport Application Service, which allows
your teams to connect to applications within private networks with fine-grained
RBAC and audit logging.

- [Dynamic App Registration](guides/dynamic-registration.mdx): Register/unregister apps without restarting Teleport.
- [High Availability Application Service Deployments](guides/ha.mdx): How to configure Teleport application access in a Highly Available (HA) configuration.
- [Protect Web Applications with Teleport](guides/connecting-apps.mdx): In this getting started guide, learn how to connect an application to your Teleport cluster by running the Teleport Application Service.
- [TCP Application Access](guides/tcp.mdx): How to configure Teleport for accessing plain TCP apps
- [VNet](guides/vnet.mdx): How to configure custom DNS zones for VNet
