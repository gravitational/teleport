---
title: Server Auto-Discovery
description: You can set up the Teleport Discovery Service to automatically enroll servers in your infrastructure.
---

The Teleport Discovery Service can connect to your cloud provider and
automatically discover and enroll servers matching configured labels. It will
then execute an install script on these discovered instances that will install
Teleport, start it and join the cluster. 

Learn how to set up auto-discovery for servers in your cloud:

{/*TOPICS*/}

- [Automatically Discover Azure Virtual Machines](servers/azure-discovery.mdx): How to configure Teleport to automatically enroll Azure virtual machines.
- [Automatically Discover GCP Compute Instances](servers/gcp-discovery.mdx): How to configure Teleport to automatically enroll GCP compute instances.
- [Configure Teleport to Automatically Enroll EC2 instances](servers/ec2-discovery.mdx): How to configure Teleport to automatically enroll EC2 instances.
