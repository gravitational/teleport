# Teleport Access Request Plugins

Plugins that integrate Teleport Access Requests with external chat and
ticketing tools. See the [official
documentation](https://goteleport.com/docs/identity-governance/access-requests/plugins/)
for setup guides.

## Supported Plugins

- [Datadog](datadog/README.md)
- [Discord](discord/README.md)
- [Email](email/README.md)
- [Jira](jira/README.md)
- [Mattermost](mattermost/README.md)
- [Microsoft Teams](msteams/README.md)
- [Opsgenie](opsgenie)
- [PagerDuty](pagerduty/README.md)
- [ServiceNow](servicenow)
- [Slack](slack/README.md)
