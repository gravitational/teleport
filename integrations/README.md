# Teleport Integrations

External integrations and plugins that extend Teleport's functionality.

## Directories

- [`access`](access/README.md) - Access request plugins for chat and ticketing tools (Slack, PagerDuty, Jira, and more)
- [`event-handler`](event-handler/README.md) - Exports Teleport audit log events to Fluentd, Datadog, Splunk, and more
- [`operator`](operator/README.md) - Kubernetes operator for managing Teleport resources via CRDs
- [`terraform`](terraform/README.md) - Terraform provider for managing Teleport resources
- [`terraform-modules`](terraform-modules/README.md) - Terraform modules for production Teleport deployments
- [`terraform-mwi`](terraform-mwi/README.md) - Terraform provider for Machine & Workload Identity ephemeral resources
